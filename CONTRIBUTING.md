# Contribuindo

Obrigado por querer contribuir com este projeto!

- Abra uma issue para discutir mudanças grandes antes de implementá-las.
- Crie um branch com nome descritivo: `feature/minha-mudanca` ou `fix/bug-descricao`.
- Envie um pull request claro, com descrição do problema e como foi resolvido.
- Respeite o `CODE_OF_CONDUCT.md`.
